

# 🚀 Nexus AI Seminar 2026 — Interactive Registration Website

A **premium, futuristic AI seminar registration website** built using **HTML, CSS, and JavaScript** with advanced UI/UX animations and real-time form handling.

---

## 🌟 Live Project Overview

Nexus AI Seminar 2026 is a modern event platform designed to provide a **smooth, interactive, and visually stunning registration experience**.

This project demonstrates:

* ✨ Advanced frontend UI/UX design
* ⚡ Smooth animations using GSAP
* 📊 Real-time form validation
* ☁️ Data storage using Google Sheets API
* 🎯 Multi-step interactive form system

---

## 📂 Project Structure

```
📁 Nexus-AI-Seminar

```

## 🔗 How Data Submission Works

* The form sends data using `fetch()` to a **Google Apps Script Web App URL**
* Data is stored directly in Google Sheets
* Uses `no-cors` mode for smooth submission
